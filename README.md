# Customer Sales Analysis

## Project Objective
Analyze customer purchasing behavior and sales trends using Pandas.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Steps Followed
1. Data loading and exploration
2. Data cleaning and preparation
3. Aggregation and customer analysis
4. Sales trend analysis
5. Pivot tables and visualization

## How to Run
1. Install requirements
2. Open customer_analysis.ipynb
3. Run all cells

## Output
- Visual dashboard
- Business insights
- Final analysis report
